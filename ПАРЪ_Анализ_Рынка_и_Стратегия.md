| Показатель | Данные | Источник |
|-----------|--------|----------|

**Вывод:** Россияне продолжают активно покупать премиум. При этом модель покупок меняется — чаще, но на меньшие суммы. Это хорошо для нас: наш средний чек (8 000—15 000 ₽) попадает в «сладкую зону» — дорого, но не шокирующе.

## 2. Целевой рынок

### 2.1. TAM → SAM → SOM (уточнённые расчёты)

```
TAM (рынок банных аксессуаров + подарков, РФ)
  = ~18–22 млрд ₽/год
  Включает: текстиль для бань, аксессуары, веники, косметику, подарочные наборы

SAM (премиум-сегмент, Москва + онлайн + корпоративный)
  = ~2–4 млрд ₽/год
  Включает: подарочные наборы от 5 000 ₽, корпоративные заказы, термы

SOM (реалистичная доля за первые 6 месяцев)
  = 3–7 млн ₽
  При: 200–400 B2C-заказов + 5–10 B2B-контрактов
```

### 2.2. Почему SOM реалистичен

- **3,8 млн** посещений терм в МСК — даже 0,01% конверсия = 380 потенциальных клиентов
- **2+ млн** уникальных банщиков только в Москве
- Персонализация — тренд, за который **63%** готовы переплатить
- Пустая ниша: ни один конкурент не имеет онлайн-конструктора

### 2.3. Сезонность (детальная для 6-месячного горизонта)

| Месяц | Индекс спроса | Фокус | Действия |

## 4. Глубокая сегментация аудитории

### 4.1. Карта сегментов

```
                     ЧАСТОТА ИСПОЛЬЗОВАНИЯ БАНИ
                     Редко ← — — — — → Регулярно
                     
  БЮДЖЕТ    ↑   [С3]               [С1]
  Высокий   |   VIP-подарок        Банный эстет
            |   20 000–35 000 ₽    12 000–20 000 ₽
            |   
            |   [С2]               [С4]
  Средний   |   Подарок со смыслом Практичный банщик
            |   7 000–15 000 ₽     5 000–8 000 ₽
            ↓
```

### 4.2. Сегмент 1: «Банный эстет» — ядро аудитории

| Параметр | Описание |
|----------|----------|
| **Демография** | Мужчина, 35–55 лет, доход 200 000+ ₽/мес |
| **Психография** | Ценитель качества, «если делать — то хорошо». Не хвастается, но разбирается. Веники сам вяжет или знает, где правильные. |
| **Поведение** | Баня 2–4 раза в месяц. Своя или клубная. Уже пользуется войлочными шапками, но «не нашёл нормального комплекта». |
| **Боль** | «Покупал на WB — развалилось после второй бани. На Ярмарке мастеров — ждал месяц, пришло не то.» |
| **Триггер покупки** | Увидел качественный конструктор → «О, тут можно меринос выбрать и вышивку сделать» |
| **Средний чек** | 12 000–20 000 ₽ |
| **Доля выручки** | ~35% |
| **LTV** | 25 000–40 000 ₽ (повторные заказы + подарки друзьям) |
| **Каналы** | Яндекс Директ, Telegram-каналы про баню и здоровье, сарафан |
| **Ключевые слова** | «банная шапка войлок премиум», «банный комплект ручной работы», «шапка для бани меринос» |

### 4.3. Сегмент 2: «Подарок со смыслом» — самый массовый

| Параметр | Описание |
|----------|----------|
| **Демография** | Женщина или мужчина, 28–50 лет, доход 100 000+ ₽/мес |
| **Психография** | Ищет подарок, который удивит. Устал(а) от стандартных «носки + пена для бритья». Хочет что-то с душой, именное. |
| **Поведение** | Гуглит «подарок мужу на день рождения оригинальный», «подарок мужчине который всё имеет». Сравнивает 5-7 вариантов перед покупкой. |
| **Боль** | «Мужу ничего не нужно, он сам всё покупает. Но он любит баню — а нормального подарочного набора не существует.» |
| **Триггер покупки** | Именная вышивка + красивая упаковка + «вау, конструктор!» |
| **Средний чек** | 7 000–15 000 ₽ |
| **Доля выручки** | ~40% |
| **LTV** | 15 000–25 000 ₽ (повторные подарки к разным датам) |
| **Каналы** | Яндекс Директ (подарочные запросы), Instagram, VK, Telegram |
| **Ключевые слова** | «подарок мужу банщику», «банный набор в подарок», «именная шапка для бани», «подарок на 23 февраля оригинальный» |

### 4.4. Сегмент 3: «VIP-подарок» — максимальный чек

| Параметр | Описание |
|----------|----------|
| **Демография** | 30–55 лет, руководитель или топ-менеджер, доход 300 000+ ₽/мес |
| **Психография** | Подарок для босса, партнёра, важного клиента. Ценник вторичен — главное впечатление и статус. |
| **Поведение** | Часто делегирует выбор ассистенту. Решение принимается за 1–2 дня. Может заказать 3–5 штук сразу. |
| **Боль** | «Нужно что-то солидное, не банальное, чтобы человек оценил. Не хочу ещё один виски / часы / кожаный портфель.» |
| **Триггер покупки** | Кейс «Абсолют» + кожаная сумка + именная гравировка = «идеально» |
| **Средний чек** | 20 000–35 000 ₽ |
| **Доля выручки** | ~10% |
| **Каналы** | Рекомендации, партнёрства с бизнес-клубами, LinkedIn/VC.ru |
| **Ключевые слова** | «VIP подарок мужчине», «подарок руководителю», «элитный подарок для бани» |

### 4.5. Сегмент 4: «Корпоративный клиент» — B2B

| Параметр | Описание |
|----------|----------|
| **Демография** | HR-менеджер, админ-отдел, маркетолог, 28–45 лет |
| **Психография** | Задача — выбрать подарки для 20–200 человек. Бюджет определён. Нужен вау-эффект при адекватной цене. |
| **Поведение** | Сравнивает 3–4 подрядчика. Запрашивает КП. Оценивает: сроки, кастомизацию, логистику. Начинает поиск за 1–2 месяца до события. |
| **Боль** | «Каждый год одно и то же — кружки, ручки, блокноты. Все сотрудники закатывают глаза. Нужно что-то функциональное и запоминающееся.» |
| **Триггер покупки** | Персонализация (имя каждого сотрудника) + лого компании + разумная цена при объёме |
| **Средний заказ** | 30–100 шт × 4 000–8 000 ₽ = 120 000–800 000 ₽ |
| **Доля выручки** | ~15% (растёт с масштабом) |
| **Каналы** | Холодные email-рассылки, LinkedIn, тендеры, партнёрства с gift-агентствами |
| **Ключевые слова** | «корпоративные подарки с логотипом», «подарки сотрудникам оптом», «персонализированные подарки для бизнеса» |

## 5. Позиционирование бренда

### 5.1. Формула позиционирования

```
Для [людей, которые ценят банную культуру и/или ищут осмысленный подарок],
ПАРЪ — это [мануфактура персональных банных комплектов],
которая [даёт собрать свой уникальный комплект в онлайн-конструкторе 
из натуральных материалов с именной вышивкой],
в отличие от [масс-маркет наборов и стандартных подарков],
потому что [каждый комплект создаётся под конкретного человека — 
это не товар, а вещь с характером].
```

### 5.2. Пирамида бренда

```
                    МИССИЯ
              «Вернуть банной 
             культуре достоинство»
                     ↓
                  ОБЕЩАНИЕ
        «Вещи, которые хочется передать»
                     ↓
               ЭМОЦИОНАЛЬНАЯ
                 ВЫГОДА
         Гордость · Уважение к себе
         Радость дарения · Ритуал
                     ↓
            ФУНКЦИОНАЛЬНАЯ ВЫГОДА
        Натуральные материалы · Комфорт
        Долговечность · Персонализация
                     ↓
            АТРИБУТЫ ПРОДУКТА
     Войлок · Меринос · Лён · Бамбук · Кожа
     Вышивка золотом · 7 предметов · Упаковка
     Онлайн-конструктор · Логотип ПАРЪ
```

### 5.3. Tone of Voice

| Параметр | Описание | Пример |
|----------|----------|--------|
| **Тон** | Спокойный, уверенный, тёплый | Не «ЛУЧШИЕ ШАПКИ!!!» а «Войлок ручной валки. 5 мм натуральной шерсти.» |
| **Стиль** | Мастер, который знает своё дело | «Меринос — самая тонкая шерсть в мире. Мы используем волокно 19.5 микрон.» |
| **Запрет** | Агрессивные скидки, капслок, «АКЦИЯ» | Никогда: «СКИДКА 50%!!! ТОЛЬКО СЕГОДНЯ!!!» |
| **Разрешено** | Спецпредложения через ценность | «Первый заказ: бесплатная вышивка при комплекте от 3 предметов» |
| **Обращение** | На «вы», без панибратства | «Выберите, что вам ближе» — не «Хватай пока не разобрали!» |
| **Юмор** | Тонкий, через предмет | «Колпак — классика. Будёновка — для тех, кто и в бане командует.» |
