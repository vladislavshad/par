# Доработка конфигуратора — ПАРЪ Мануфактура
**Дата:** 2026-04-10
**Сайт:** par-mfg.vercel.app/constructor
**Стек:** Next.js, React, Tailwind CSS

## Текущее состояние

Конфигуратор работает как 4-шаговый визард:
1. Выбор предметов (7 позиций, от 1,400 до 5,900₽)
2. Кастомизация (материалы, цвета, размеры)
3. Упаковка
4. Оформление заказа

## Выявленные проблемы

### Критические (влияют на конверсию)

**1. Нет живого подсчёта цены**
- Пользователь не видит итог пока выбирает предметы
- Нужен sticky-бар внизу: "Итого: X₽ | Y предметов"
- При добавлении/удалении — анимация изменения цены

**2. Нет превью кастомизации до входа в шаг 2**
- Пользователь не понимает, что "кастомизация" = выбор материала, цвета, вышивки
- Нужен подзаголовок под каждым товаром: "войлок / меринос / альпака • 4 цвета • вышивка"
- Или мини-карусель вариантов при наведении

**3. Вышивка — главный UTP, но упоминается только в заголовке**
- "Именная вышивка золотом" — ключевое преимущество, но пользователь не видит примеров
- Нужен интерактивный превью: поле ввода имени → моментальный рендер на шапке/полотенце
- Это killer-feature, она должна быть на шаге 1

**4. Нет визуальной обратной связи при выборе**
- Непонятно, что выбрано, а что нет
- Нужны: чекбоксы/тоглы, подсветка карточки, значок ✓, счётчик

### Важные (улучшают UX)

**5. Нет готовых пресетов "Классика / Мастер / Абсолют"**
- Пользователь уже знает наборы из каталога, но в конструкторе начинает с нуля
- Нужны кнопки "Начать с набора" → предзаполнение выбранных предметов
- Сокращает путь к покупке на 2-3 шага

**6. Нет индикации шагов и прогресса**
- Прогресс-бар невидим или неинтуитивен
- Нужен горизонтальный степпер: ①Предметы → ②Материалы → ③Упаковка → ④Заказ
- С возможностью кликнуть назад на любой шаг

**7. Нет сравнения материалов**
- При выборе материала (войлок vs меринос vs альпака) нет объяснения разницы
- Нужна таблица: толщина, мягкость, теплозащита, цена, фото текстуры

**8. Mobile UX**
- Проверить, что на мобильных карточки товаров не обрезаются
- Sticky-кнопка "Далее" внизу экрана на мобильных
- Свайп-карусель для вариантов материалов

### Дополнительные (nice-to-have)

**9. Шаринг конфигурации**
- Кнопка "Поделиться набором" → уникальная ссылка
- Для подарков: "Отправь ссылку — пусть сам выберет материал"

**10. Сохранение конфигурации**
- LocalStorage: сохранять выбор при закрытии страницы
- "Продолжить с последнего раза" при повторном визите

**11. Визуализация набора**
- 3D или flat-lay превью собранного набора
- Обновляется в реальном времени при каждом выборе

**12. SEO и sharing**
- Мета-теги для шаринга конфигурации в соцсетях
- OG-image с превью собранного набора
