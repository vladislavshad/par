# ПАРЪ — Премиальные банные комплекты

Онлайн-конструктор персональных банных комплектов ручной работы. Натуральные материалы, именная вышивка, элитная упаковка.

**Production:** https://par-sage.vercel.app

## Возможности

- **Лендинг** — презентация бренда с видео-контентом
- **Конструктор комплекта** — 4-шаговый визард: выбор предметов, кастомизация, упаковка, оформление заказа
- **Конструктор шапки** — визуальный конфигуратор: 4 формы (колпак, будёновка, ушанка, панама), 3 цвета, 4 материала, превью вышивки
- **7 предметов** — шапка, полотенце, халат, килт, подстилка, тапочки, сумка
- **Именная вышивка** — монограмма, имя или фраза, 4 шрифта, 7 цветов нити, логотип ПАРЪ
- **Упаковка** — от стандартной крафт-коробки до VIP кожаного кейса (льняная сумка бесплатно от 3 предметов)
- **Кастомные заказы** — форма связи для корпоративных и нестандартных запросов
- **Админ-панель** — управление заказами на `/admin`
- **Telegram-уведомления** — заказы отправляются в Telegram-бот
- **E2E-тесты** — Playwright-тесты конфигуратора шапок

## Стек

- **Next.js 16** (App Router, Turbopack)
- **React 19**
- **Tailwind CSS 4**
- **Zustand** — состояние конструктора с persist (localStorage)
- **Framer Motion** — анимации
- **Playwright** — E2E-тестирование
- **Vercel** — хостинг и деплой

## Запуск

```bash
npm install
npm run dev
```

Открыть http://localhost:3000

## Структура

```
src/
  app/
    page.tsx                  # Лендинг
    constructor/page.tsx      # Конструктор комплекта
    constructor/hat/page.tsx  # Конструктор шапки
    admin/page.tsx            # Админ-панель
    api/order/                # API заказов
    api/admin/                # API админки
  components/
    landing/                  # Компоненты лендинга
    constructor/              # Компоненты конструктора
      HatPreview.tsx          # Превью шапки с предзагрузкой и анимацией
      HatOptionsPanel.tsx     # Панель опций шапки
      StickyPriceBar.tsx      # Фиксированная панель цены
      OrderSummary.tsx        # Итог заказа
  data/products.ts            # Каталог продуктов, материалов, цен
  store/useConstructor.ts     # Zustand стор с persist
  lib/                        # DB, Telegram
e2e/
  hat-configurator.spec.ts    # E2E-тесты конфигуратора шапок
public/
  images/                     # Фото продуктов
  images/hats/base/           # Базовые фото шапок (форма × цвет × материал)
  images/hats/logo/           # Фото с логотипом ПАРЪ (форма × тон × нить)
  images/hats/engraving/      # Фото с вышивкой (форма × тон × нить × позиция)
  videos/                     # Видео для лендинга
```

## Переменные окружения (опционально)

```
POSTGRES_URL=           # Vercel Postgres для хранения заказов
TELEGRAM_BOT_TOKEN=     # Telegram бот для уведомлений
TELEGRAM_CHAT_ID=       # Chat ID для отправки заказов
ADMIN_PASSWORD=         # Пароль для админ-панели
```
